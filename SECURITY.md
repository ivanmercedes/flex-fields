# Security Policy

## Supported Versions

Please see the table below to check which versions are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an e-mail to the maintainer rather than creating a public issue. All security vulnerabilities will be promptly addressed.

We will try to acknowledge receipt of the vulnerability within 24 hours and will send regular updates about our progress. If you haven't received a reply within 24 hours, please follow up to ensure we received your message.
